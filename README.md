# Gator
Aka blog-aggregator.
A multi-user CLI blog aggregator written in Go.

## Prerequisites

- Go 1.22+
- PostgreSQL

## Setup

1. Create a config file at `~/.gatorconfig.json`:

   ```json
   {
     "db_url": "postgres://user:password@localhost:5432/gator?sslmode=disable"
   }
2. go install
